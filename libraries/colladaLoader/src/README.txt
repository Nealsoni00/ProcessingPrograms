Compiling:
---------
to compile the sources additional libs is needed:

- core.jar (copy from Processing "core" folder)

- Jogl libraries (depends on your platform)
	- gluegen-rt *.* (2 jars: classes and natives)
	- jogl-all *.* (2 jars: classes and natives)
	( downloadable on http://www.jogamp.org/ or copy from
	 Processing package ("core" folder)
